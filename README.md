## Installation

### Step 1
Clone this repo\
`cd` to it
```
git clone https://github.com/10Keshav/dotfiles.git
cd dotfiles
```
Use stow 
```
stow .
```

### Step 2
Check out [package_list.txt](package_list.txt) for all the packages you need installed.\
You can:
#### Use AUR (yay or paru)
```
yay -S $(cat package_list.txt)
```
